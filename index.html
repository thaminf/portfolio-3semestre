O que é Javascript?

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. Ela é comumente usada para criar páginas web interativas, sendo executada no navegador do usuário.

Variáveis

A variável serve para conseguirmos lidar com dados sem saber exatamente o que esse dado é, ou seja, esse dado vai variar mas poderemos lidar com ele. Não se pode ter dois dados dentro de uma variável ao mesmo tempo. De forma mais teórica, variáveis são espaços de armazenamento nomeados que contêm dados ou referências para dados. Em JavaScript, você pode declarar variáveis usando var (antigo), let (variável que pode ser reatribuída) ou const (constante, não pode ser reatribuída).

Tipos de Dados

Boolean, Null, Undefined, Number e String.

String: String é uma cadeia de caracteres. Palavras, frases, textos inteiros são cadeia de caracteres. Um desses caracteres é o espaço/caracter nenhum.

Undefined: Quando não inicializamos a variável e o programa não sabe o que tem ali dentro, diferente do null.

Null: Certeza de que não tem nada dentro de uma variável. Que a variável está vazia.

Boolean: true ou false, que permite fazer operações lógicas.

Operadores Matemáticos

JavaScript oferece uma variedade de operadores matemáticos para realizar operações em números. Isso inclui operadores aritméticos como adição, subtração, multiplicação, divisão e módulo.

Funções

Funções em JavaScript são blocos de código reutilizáveis que podem receber parâmetros, realizar uma tarefa e retornar um valor. Elas são fundamentais para organizar e modularizar o código pois são blocos pequenos.

Estruturas condicionais e de repetição

Estruturas condicionais, como if, else if e else, permitem controlar o fluxo de execução do programa com base em condições lógicas. Isso é fundamental para a tomada de decisões no código.

As estruturas de repetição, como for, while e do-while, permitem que você execute um bloco de código várias vezes. Isso é útil para automatizar tarefas repetitivas.

Objetos e Arrays

Objetos são estruturas de dados que representam entidades com propriedades e métodos associados. Arrays são estruturas que armazenam elementos indexados. Ambos são essenciais organizar e manipular dados em JavaScript.

Math e Métodos Nativos

O objeto Math fornece funções matemáticas, como random para gerar números aleatórios e round para arredondar números. Além disso, JavaScript oferece diversos métodos nativos para manipular strings, arrays e números.
